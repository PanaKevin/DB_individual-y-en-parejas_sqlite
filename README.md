# Bases de Datos en JSON

Este repositorio contiene los archivos JSON y los archivos .bd (formato permitido por SQLite) correspondientes a dos bases de datos:

- **Base de datos escolar**: información relacionada con la gestión académica (estudiantes, docentes, calificaciones, etc.).
- **Base de datos para la biblioteca virtual Miguel de Cervantes**: información relacionada con la gestión de una biblioteca digital (libros, usuarios, préstamos, categorías, etc.).
